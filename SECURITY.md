# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Civic Cycle, please report it responsibly.

**Email:** security@civiccycle.org

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge receipt within 48 hours and aim to provide a fix within 7 days for critical issues.

## Scope

This policy applies to the Civic Cycle web application and its infrastructure.
